# 03: AI Travel Itinerary Generator

This project is an AI-powered workflow built using **n8n** and the **Groq API** that generates personalized travel itineraries based on user preferences.

The workflow exposes a **Webhook endpoint** that accepts travel details through an HTTP POST request. After receiving the request, n8n extracts the required information, sends it to an AI Agent powered by **Groq's Llama 3.3 70B Versatile** model, and converts the generated itinerary into a downloadable text file.

---

## 🎯 Project Objective

The objective of this project is to understand how to integrate Large Language Models (LLMs) into workflow automation using n8n.

Through this project, I gained hands-on experience with:

- Creating AI-powered workflows in n8n
- Receiving user input through HTTP Webhooks
- Extracting and mapping incoming request data
- Integrating the Groq API with n8n
- Generating structured AI responses
- Converting AI-generated content into downloadable files

---

## ⚙️ Workflow Overview

```text

Webhook (POST Request)
        │
        ▼
Edit Fields
        │
        ▼
    AI Agent
(Groq Chat Model)
        │
        ▼
Convert to File
```

---

## 📝 Workflow Explanation

### 🔹 Webhook

The Webhook node creates an HTTP endpoint that listens for incoming **POST** requests. When a request containing travel details is received, it triggers the workflow and passes the request payload to the next node.

**Example Request**

```json
{
  "destination": "Manali",
  "days": 4,
  "budget": 18000,
  "transport": "Bus"
}
```

---

### 🔹 Edit Fields

The Edit Fields node extracts the required values from the incoming request and prepares them for the AI Agent.

The extracted fields include:

- `destination`
- `days`
- `budget`
- `transport`

---

### 🔹 AI Agent (Groq)

The AI Agent uses **Groq's Llama 3.3 70B Versatile** language model to generate a personalized travel itinerary based on the provided travel preferences.

The generated itinerary includes:

- Trip Overview
- Transportation Recommendation
- Day-wise Itinerary
- Top Attractions & Activities
- Recommended Local Foods
- Budget Breakdown
- Travel Tips
- Best Time to Visit
- Packing Suggestions
- Trip Summary

---

### 🔹 Convert to File

The Convert to File node transforms the AI-generated itinerary into a plain text (`.txt`) file, making it easy to download, share, or integrate into other workflows.

---

3. Activate the workflow.

5. Send a POST request using Postman or any HTTP client.
